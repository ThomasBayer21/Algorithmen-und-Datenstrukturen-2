Tabellengestützte Programmierung

Kapitel 5 · Editierdistanz & Blocksatz
Tabellengestützte Programmierung (Dynamic Programming) löst Probleme, indem Teillösungen in einer Tabelle gespeichert und wiederverwendet werden.
Editierdistanz: Minimale Anzahl von Einfüge-, Lösch- und Ersetz-Operationen, um einen String in einen anderen umzuwandeln. Jede Zelle D(i,j) ergibt sich aus den drei Nachbarzellen – Laufzeit O(|s|·|t|).
Blocksatz-DP: Optimale Zeilenumbrüche für einen Text, sodass der quadratische Straf-Malus für überschüssigen Leerraum minimiert wird – Laufzeit O(n²).
Eingabe
Formel
D(i,j) = min(
  D(i−1,j)+1   [Löschen]
  D(i,j−1)+1   [Einfügen]
  D(i−1,j−1)+δ   [δ=0 gleich, 1 ersetzen]
)
